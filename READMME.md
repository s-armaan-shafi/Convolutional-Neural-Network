# Fashion MNIST Image Classification using CNN

This project implements a Convolutional Neural Network (CNN)
to classify Fashion-MNIST images into 10 categories.

## Dataset
Fashion-MNIST (28Ã—28 grayscale images)

## Model
- 2 Convolution layers
- MaxPooling
- Dense + Dropout
- Softmax output

## Accuracy
~90% on test data

## How to Run
pip install -r requirements.txt  
python fashion_cnn.py
